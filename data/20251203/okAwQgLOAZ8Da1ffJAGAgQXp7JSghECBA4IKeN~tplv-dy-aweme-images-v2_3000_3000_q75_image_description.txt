### **二、详细内容解析**

#### **第四部分：串联超前校正**

**标题：四、串联超前校正**

##### **1. 性能目标**
- “最多提高60°” —— 指的是相位裕度可以提升的最大角度。

##### **2. 公式列表**
1. **φm = γ* - γ + (5~10)°**  
   - φm 是最大超前相位角。
   - γ* 是期望的相位裕度。
   - γ 是原系统的相位裕度。
   - 补偿后增加5到10度的安全余量。

2. **φm = arcsin[(a-1)/(a+1)] → a = (1 + sinφm)/(1 - sinφm)**  
   - 通过相位裕度计算参数 a（即超前校正器的比例系数）。
   - 这是超前校正器设计中的关键关系式。

3. **20log G(jωc) + 10log a = 0 → ωc'**  
   - 调整增益使新的截止频率 ωc' 满足闭环系统要求。
   - 用于求解校正后的截止频率。

4. **ωc' = 1/(T√a) → T**  
   - 由新截止频率反推出时间常数 T。

##### **3. 性能影响分析（蓝色文字）**
- **稳定性**：→ **延时量降低**，σ% = 1/(1+sinγ*)  
  - 稳定性增强，因为相位裕度增大，系统响应更快，振荡减小。
- **准确性**：→ 与校正的 k 有关，k₀₁，稳态误差降低  
  - 若 k < 1，则稳态误差变大（需注意设计时选择合适的 k）。
- **快速性**：→ ωc 增大，快速性变好  
  - 截止频率升高意味着系统响应速度加快。
- **高频段抗干扰能力**：→ 变差  
  - 因为超前校正会放大高频信号，导致抗干扰能力下降。

#### **第五部分：串联滞后校正**

**标题：五、串联滞后校正**

##### **1. 公式列表**
1. **γ = γ* + 6° → ωc'**  
   - 设计目标是让相位裕度略高于期望值（加6度安全余量），以确定新的截止频率。

2. **20log G(jωm) + 20log b = 0 → b**  
   - 利用中频段增益调整来确定滞后校正器的衰减因子 b。
   - 目的是在不改变相位的前提下，降低增益。

3. **1/(bT) = ωm/10 → T**  
   - 根据转折频率 ωm 和 b 的关系，求出时间常数 T。

##### **2. 性能影响分析（蓝色文字）**
- **稳定性与准确性**：→ 同超前校正，两者同步改善  
  - 滞后校正通常用于提升低频增益，从而改善稳态精度，同时保持稳定。
- **快速性**：→ 变差  
  - 因为滞后校正降低了截止频率，响应速度变慢。
- **高频段抗干扰能力**：→ 提高  
  - 滞后校正具有低通特性，抑制高频噪声，提升抗干扰能力。
