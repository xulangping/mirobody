### **一、顶部公式与定义**
1. **谐振频率公式**：
   $$
   \omega_r = \omega_n \sqrt{1 - 2\xi^2}, \quad 0 < \xi \leq \frac{\sqrt{2}}{2}
   $$
   - 描述了二阶系统在阻尼比 $\xi$ 范围内出现谐振时的频率。

2. **谐振峰值公式**：
   $$
   M_r = A(\omega_r) = \frac{1}{2\xi\sqrt{1 - \xi^2}}, \quad 0 < \xi \leq \frac{\sqrt{2}}{2}
   $$
   - 表示系统在谐振频率下的幅值增益。

3. **特殊条件说明**：
   - 当 $\xi = \frac{\sqrt{2}}{2}$ 时，$M_r = 1$，且 $\omega_r$ 和 $M_r$ 都是 $\xi$ 的最小值。
   - 当 $0 < \xi < \frac{\sqrt{2}}{2}$ 时：
     - $0 < \omega < \omega_r$：$A(\omega)$ 单调递增；
     - $\omega > \omega_r$：$A(\omega)$ 单调递减；
     - 当 $\frac{\sqrt{2}}{2} < \xi < 1$：$A(\omega)$ 单调递减。

### **二、图5-10：典型环节幅相特性曲线**
该图展示了三种典型环节在复平面上的奈奎斯特（Nyquist）曲线：

#### **(a) 比例环节与积分/微分环节**
- **比例环节**：$K>0$ 和 $K<0$ 的情况，对应实轴上的点。
- **积分环节**：$\frac{1}{j\omega}$，轨迹为从正虚轴到负虚轴的半圆。
- **微分环节**：$jT\omega$，轨迹为从原点出发沿正虚轴延伸。

#### **(b) 惯性一阶微分环节**
- 传递函数形式：$\frac{1}{1 + jT\omega}$ 或 $1 + jT\omega$。
- 曲线为单位圆的一部分，分别表示最小相位和非最小相位特性。

#### **(c) 振荡二阶微分环节**
- 传递函数形式：$\frac{1}{1 - \frac{\omega^2}{\omega_n^2} + j2\xi\frac{\omega}{\omega_n}}$ 及其变体。
- 曲线呈“8”字形或双叶状，体现振荡特性的相位变化。

> 图注：“图5-10 典型环节幅相特性曲线”

### **三、图5-11：典型环节对数频率特性曲线（伯德图）**
此图展示了上述环节的对数幅频和相频特性：

#### **(a) 比例环节与积分/微分环节**
- 幅频特性：斜率为0 dB/dec（比例），-20 dB/dec（积分），+20 dB/dec（微分）。
- 相频特性：分别为0°、-90°、+90°。

#### **(b) 惯性一阶微分环节**
- 幅频特性：低频段平坦，转折频率后下降或上升。
- 相频特性：从0°过渡到-90°（惯性）或+90°（微分）。

#### **(c) 振荡二阶微分环节**
- 幅频特性：存在峰值（谐振峰），取决于阻尼比。
- 相频特性：从0°逐渐变为-180°，中间有拐点。

> 图注：“图5-11 典型环节对数频率特性曲线（伯德图）”
